Appliquer à Claude Code

Écrire les services gérés dans ~/.claude.json. Chaque entrée est un appel mcpwarden run — aucun secret ni référence n'est écrit dans le fichier ; le secret est résolu depuis le coffre au lancement. Une sauvegarde horodatée est créée et le reste de votre configuration est préservé.
